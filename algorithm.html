This post is my notes for preparing a coding interview, including some thoughts about classic problems, and some terminologies that might be necessary for an interview.

1 binary operations

The basic binary operations are as follows

<< left shift 1<<2 gets 2
>> right shift 4>>2 gets 1
& bitwise and 0 & 1 = 0
\vert bitwise or  
\~ bitwise complement unsigned int max=\~0
^ bitwise Exclusive-Or (XOR) 10^11=01

1.1 set some bit

x |= 1 << n;

1.2 reset some bit

x &= (~0 ^ (1<<n))

1.3 check some bit

x & (1<<n)

1.5 storeage of negative numbers

A positive number is representated as itself while a negative number is representated as the two's complement of its absolute value.

In other words, the binary representation of -K as a N-bit number is concat(1, 2(N-1)-K).
